Frequently Asked Questions

What kind of files can APKScan scan?

APKScan can only scan and detect Android Application Package files (.apk).

What is maximum size of file that can be sent to APKScan?

Maximum file size is 200 mb.

Why is Scan button grayed out? (I can't click Scan button.)

You must agree with our terms of service before scanning.

How long does a scan last?

Scan time varies from one second to a few minutes depending on your position in the queue. If we already scanned the file you are uploading you will instantly see results.

How do you scan .apk files?

When we receive your .apk file we install it to Android virtual machine. Then we wait for the results of installed antivirus apps.
